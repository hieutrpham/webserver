What this site verifies

Coverage designed around the Webserv subject

The site does not only show pages. It gives each route a clear purpose so failures can be traced to a specific server feature.

  • Multiple MIME types and static file delivery
  • Index file lookup and trailing-slash redirects
  • Longest matching location selection
  • Allowed and rejected HTTP methods
  • Multipart uploads and upload storage
  • CGI GET and POST execution
  • Generated autoindex pages and nested paths
  • Custom error responses and body-size limits
