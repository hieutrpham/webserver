About this site

This website is intended to test the features of the Webserv HTTP server. Use the navigation bar above to verify that static files, directory listings, uploads, and JavaScript are all being served correctly.

Things to try

  • Home Serves the default index.html.
  • About Tests serving another HTML page.
  • Gallery Tests serving multiple images from the /images directory.
  • Download Downloads a plain text file and verifies file serving and MIME types.
  • Autoindex Test Opens a directory without an index page. If autoindex is enabled in your configuration, the server should generate a directory listing automatically.
  • Upload Upload a file to verify HTTP POST handling and file uploads.
  • Click the button below to verify that JavaScript files are being served correctly.
